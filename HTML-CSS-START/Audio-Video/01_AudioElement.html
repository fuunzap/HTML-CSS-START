• AAC — формат сжастия с потерями, дает более высокое качество чем MP3
  • MP3 — лицензируемый формат файла для зранения аудиоинформации
  • OGG — открытый стандарт мультимедиа контейнера
  • WAV — проприетарный аудиоформат без какого-либо сжатия;
  • WEBM — открытый беслптаный медиаформат Google, основанный на аудиокодеке Vorbis
