HLS (Hue, Saturtationm Lihtness) - цветоая моель, в которой цветовыми координатами является тон, насыщенность и яркость. HLS - Википедия

  1. оттенок цвета указываетсяв градусах поворота цвтного круга( 0 градусов - красный, 120 градусов - зелёный, 240 градусов - голубой и т.д).
  2. Насышенность текста указывается в процентах (по мере понижения процентов цвет будет блекнуть).
  3. Яркость текста так же указываетмя в процентах(0% - тёмный, 100% - светлый).
